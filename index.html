Famous Cats

Этот проект представляет собой веб-приложение на Django, посвященное известным кошкам, которые стали популярными в литературе и кино.

Основные функции:

  • Просмотр списка знаменитых кошек.
  • Просмотр отдельных статей о кошках.
  • Добавление новых статей (требуется авторизация).
  • Регистрация новых пользователей.
  • Авторизация пользователей.
  • Форма обратной связи.

Учет производства металлов

Веб-приложение на Django, для учета производства и потребления разных типов металлов на металлургическом заводе.

Основные функции:

  • Просмотр результурующей таблицы
  • Добавление новой записи
  • Редактирвоание записей
  • Просмотр графика
  • Выборка по датам и по неделям
  • Учет списка контейнеров для операторов
